# 🔧 Automobile Spare Parts Management System

A full-stack MERN application for spare parts inventory 
management with real-time stock tracking and order handling.

## Tech Stack
- **Frontend:** React.js, React Router DOM
- **Backend:** Node.js, Express.js, RESTful APIs
- **Database:** MongoDB

## Features
- Real-time stock tracking
- Parts catalog with CRUD operations
- Multi-page navigation with React Router DOM
- Efficient MongoDB schema design
- Dynamic frontend-backend integration

## Setup
```bash
# Backend
Nodemon server.js

# Admin
cd admin
ng server

# Frontend
cd frontend
ng serve
```

## Project Structure
```
├── client/          # React frontend
├── server/          # Node.js + Express backend
├── models/          # MongoDB schemas
└── routes/          # API routes
```
